# CivitAI Model Downloader

Надёжная многопоточная утилита командной строки для массовой загрузки моделей, изображений и метаданных с [CivitAI](https://civitai.com) — по имени пользователя или напрямую по ID модели.

Основано на оригинальном скрипте [Confuzu](https://github.com/Confuzu), с добавлением различных функций и поддержкой [galCME](https://github.com/LeonWGal/galCME).

[English version](README.md)

## Возможности

- **Загрузка по юзернейму или по ID модели** — можно скачать всё, что выложил автор, либо запросить конкретные модели напрямую через `--model_id`.
- **Докачка прерванных загрузок** — прерванные загрузки продолжаются через HTTP `Range`-запросы, а не начинаются заново.
- **Проверка целостности файлов** — каждый файл модели сверяется с размером/SHA256-хэшем из API; повреждённые или недокачанные файлы перекачиваются автоматически. Фоновый сканер (`--deep_check`) может перепроверить всю библиотеку при каждом запуске.
- **Автоперезапуск медленных загрузок** — если скорость падает ниже 100 КБ/с дольше 30 секунд, загрузка прерывается и запускается заново.
- **Метаданные генерации для примеров изображений** — prompt, seed, sampler и модель сохраняются рядом с каждой картинкой-примером в файл `{id}_meta.txt` (или `{id}_no_meta.txt`, если данных нет).
- **Поиск и удаление дублей** — убирает повторяющиеся файлы и повторяющиеся примеры изображений (по ID картинки в CivitAI), оставляя копию с более информативным именем.
- **Файлы-маркеры `.skip`** — положите файл `.skip` в папку пользователя/модели/версии, чтобы навсегда исключить её из последующих запусков.
- **Гибкая фильтрация** — по типу контента (`Lora`, `Checkpoints`, `Embeddings`, `Training_Data`, `Other`, `All`), по исключению типа и по базовой модели (нечёткое сравнение через запятую, например `--base_models Illustrious,Pony`).
- **Защита от path traversal** — имена пользователей, файлов и папок, приходящие из API, строго очищаются, а каждый собранный путь проверяется на то, что он остаётся внутри выходной директории, прежде чем что-либо записывается на диск.
- **Безопасная работа с токеном** — токен API никогда не отображается в терминале: передавайте его через `--token`, переменную окружения `CIVITAI_API_TOKEN`, либо вводите в скрытом приглашении.
- **Устойчивость к длинным путям** — слишком длинные имена папок/файлов автоматически укорачиваются и мигрируются для совместимости с Windows.
- **Живой прогресс в консоли** — общий прогресс-бар по пользователю плюс отдельный бар "ближайшей к завершению" загрузки, и цветная итоговая сводка.

## Требования

```
pip install requests tqdm
```

Рекомендуется Python 3.8+ (используется оператор "морж").

## Использование

```bash
python civitAI_Model_downloader.py <username> [<username2> ...] [опции]
python civitAI_Model_downloader.py --model_id 12345,67890 [опции]
```

### Авторизация

Скрипту нужен API-токен CivitAI (создаётся на civitai.com → Account Settings → API Keys). Приоритет:

1. `--token ВАШ_ТОКЕН`
2. Переменная окружения `CIVITAI_API_TOKEN`
3. Скрытый интерактивный ввод (символы не отображаются)

### Опции

| Флаг | По умолчанию | Описание |
|---|---|---|
| `usernames` | — | Один или несколько юзернеймов CivitAI (позиционный аргумент, не нужен при `--model_id`) |
| `--model_id`, `--model_ids` | — | ID модели или список ID через запятую для прямой загрузки, минуя поиск по юзернейму |
| `--download_type` | `All` | Один из: `Lora`, `Checkpoints`, `Embeddings`, `Training_Data`, `Other`, `All` |
| `--exclude_type` | — | Скачать всё, кроме указанного типа (несовместимо с `--download_type`) |
| `--base_models` | — | Фильтр по базовой модели через запятую, сравнение регистронезависимое, по подстроке (например `Illustrious,Pony,SDXL`) |
| `--max_threads` | `5` | Количество параллельных потоков загрузки |
| `--max_tries` | `3` | Максимум повторов на файл |
| `--retry_delay` | `10` | Задержка между повторами в секундах (с экспоненциальным ростом) |
| `--deep_check` | выкл. | Проверка SHA256-хэшей всех существующих файлов при запуске (медленно, но надёжно) |
| `--token` | — | API-токен (лучше использовать переменную окружения) |

### Примеры

```bash
# Скачать все LoRA конкретного автора
python civitAI_Model_downloader.py someartist --download_type Lora

# Скачать всё, кроме чекпоинтов, у двух пользователей
python civitAI_Model_downloader.py userA userB --exclude_type Checkpoints

# Скачать две конкретные модели по ID
python civitAI_Model_downloader.py --model_id 12345,67890

# Только модели на базе Illustrious/Pony, с увеличенным числом потоков
python civitAI_Model_downloader.py someartist --base_models Illustrious,Pony --max_threads 8

# Полная проверка целостности уже скачанной библиотеки
python civitAI_Model_downloader.py someartist --deep_check
```

## Структура выходных файлов

```
model_downloads/
└── <username>/
    └── <категория>/            # Lora, Checkpoints, Embeddings, Training_Data, Other
        └── <base_model>/       # например Illustrious, Pony, SDXL 1.0 (если указано в API)
            └── <название модели>/
                └── <название версии>/
                    ├── <файл>.safetensors
                    ├── <файл>.civitai.info      # полные метаданные версии из API
                    ├── <файл>.json              # описание + trigger words
                    ├── triggerWords.txt
                    ├── <name>.preview.jpg
                    └── examples/
                        ├── <name>_<id>.jpeg
                        └── <id>_meta.txt         # либо <id>_no_meta.txt
```

Логи, сводки по пользователям и отчёты о неудачных загрузках пишутся в папку `logs/`.

## Исключение отдельных элементов

Создайте пустой файл `.skip` внутри папки `username/`, `.../<модель>/` или `.../<модель>/<версия>/`, чтобы скрипт навсегда пропускал этого пользователя, модель или версию при последующих запусках.

## Заметки по безопасности

- API-токен передаётся только через заголовок `Authorization`, никогда не добавляется в URL.
- Все имена файлов и юзернеймы, приходящие из ответов API, очищаются и собираются в путь с проверкой на выход за пределы базовой директории (`safe_path_join`), прежде чем скрипт обратится к файловой системе.
- Пагинация переходит только по `nextPage`-ссылкам с хостом `civitai.com` / `www.civitai.com` — это защищает от утечки токена при подделанном или скомпрометированном ответе API.
